Installing with MultiMC (recommended)

MultiMC allows you to have completely seperate Minecraft instances, each with their own mods, worlds, etc. Once setup is done, it is super easy to use and add to, and for that reason I highly recommend using it.


MultiMC Download and Install

To download MultiMC, go to the MultiMC website and scroll down to the Download & Install section. Click on the download that corresponds to your OS.


MultiMC Setup

After MultiMC is opened for the first time, a setup screen will appear. Choose your language, then click Next.

If the next screen is empty or you do not have Java 17, you can download it from Adoptium.

Once Java is installed, click Refresh, and then click on the first item in the list. On the bottom of the screen, change the maximum memory allocation to at least 2048 MiB. Click Next. If a screen appears asking if you want to send analytics, you can disable it (if you want) or just click on Finish.


Setting Up Fabric

You should now be in the main menu. Click on "Add Instance". Next, click on 1.19 (NOT 1.19.1), then click OK.

Back in the main menu, click on Edit Instance. Install Fabric by clicking on the button on the right, followed by clicking OK. Click on Install mods (below Install Fabric) to view the mod menu.

Download the Fabric API mod (for version 1.19) and the Simple Voice Chat mod (the one labeled [FABRIC][1.19]). Once downloaded, drag and drop them into the Loader mods window.


Transferring Settings

MultiMC's instances are seperated from the main Minecraft folder, so you will need to transfer your settings over. Click on "Minecraft Folder" on the right side of the MultiMC window. Next, run the keyboard shortcut Copy and paste the following folders from your AppData .minecraft folder to your MultiMC .minecraft folder:

  • resourcepacks
  • shaderpacks (if folder exists)
  • saves
  • options.txt / options
  • saves
Minecraft is now almost ready to launch. Before you can launch it, you need to add an account. Log in with your Microsoft/Mojang account by clicking on "Add Microsoft" or "Add Mojang", and you should be done!


OptiFine and Other Mods

Most mods work well with Fabric and MultiMC, and have an available download for Fabric. However, OptiFine does not by default.

I do not recommend using OptiFine with Fabric, as it can be buggy and lead to crashes. Instead, I suggest trying the following group of mods out first, as it has similar performance and features but will be much less buggy. As always, download the Fabric version for 1.19.

  • Animatica (animated textures)
  • CIT Resewn (custom item textures)
  • Continuity (connected textures)
  • Indium (required to make some of these mods work)
  • Iris (allows you to use optifine shaders)
  • LambDynamicLights (dynamic lighting like optifine's)
  • Logical Zoom (replacement for optifine zoom)
  • More Block Predicates (block models can change based on conditions like biome, light, etc.)
  • Mod Menu (adds a menu of mods, useful for configuration)
  • Sodium (major optimization, highly recommended)
  • Reese's Sodium Options (changes the sodium options menu)
  • Sodium Extra (adds a lot of features that optifine has but sodium doesn't have)

If you do need to use OptiFine, you need to use the OptiFabric mod with it for it to properly function with Fabric. Install both jar files like you would for any other mod.

Giving the mod microphone access (Mac OS)

Try doing this if Minecraft doesn't have access to the microphone. Taken from the Simple Voice Chat Github page:

Right click on the MultiMC app icon, and click on Show in Finder. Right click on the app icon in Finder. Hold down the option key, and click on "Copy MultiMC as pathname".

Open Terminal and type in codesign --force --deep --sign - pathname, but replace pathname with the path you just copied. Press enter. After restarting Minecraft, it should have access to the microphone.

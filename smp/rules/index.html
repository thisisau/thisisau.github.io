SMP Rules (toggle light mode)

Anything not listed in the rules is generally okay, however common sense should still be used. If you have any questions, ask a moderator. These rules may change, and if they do, an announcement will be posted in this server.

If these rules are infringed upon, punishments will be issued based on the severity of the act.


No cheating

This rule includes:

  • Using cheat clients
  • Using a resource pack to see through blocks that would normally be fully opaque (xray)
  • Creating a new world with the same seed to see things you couldn't see before
  • Using duplication glitches that involve inventories or item entities
  • Using the world seed to find strongholds, end cities, woodland mansions, or ocean monuments (note: the world seed is currently unavailable)

This rule does not include:

  • Using duplication glitches that involve non-tile entities placed in the world (e.g rail dupes, falling block duplication)
  • Using the world seed to find any structures or world features not listed above


No toxicity

These rules are loosely defined and should be up to people to decide how they want to follow them in specific scenarios. For example, during a war, griefing and stealing is up to the people who are in the war.

  • Don't start a conflict with someone who isn't trying to be in a conflict.
  • Don't try to grief other people's areas. If an explosion happens by accident, try to patch it up or notify them. Messing with other people's bases in ways that don't destroy much property and are just for fun is okay.
  • Don't take items from people unless they give you permission. If it is a small amount of a non-valuable item, taking it is fine, as long as you notify them and give the item back if they ask.
  • Don't steal from farms, even if you were loading that farm. Only use farms that aren't yours in ways that you have been given permission for.
  • Don't kill people if they don't want you to.
  • Don't be toxic to others, may it be through chat or actions on other players.


Communal Rules: Recommended/Optional

  • Bases are recommended to be within 1000 blocks of spawn.
  • There is a warzone from the coordinates (250, 250) to (500, 500) where pvp, griefing, and stealing are allowed.
  • There are recommended server sessions from to every day.


Communal Rules: Gear (Required)

  • Armor: All non-netherite armor and all enchantments are allowed.
